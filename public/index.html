Available API Endpoints

System
GET /api/health
Health check endpoint - returns system status and version
GET /api/dashboard
Get complete dashboard data including sentiment, sources, and recent batches
Trackers
GET /api/trackers
List all configured trackers
GET /api/trackers/:id
Get detailed information about a specific tracker
GET /api/trackers/:id/sentiment
Get sentiment aggregates for a tracker (supports time bucket filtering)
Sources
GET /api/sources
List all configured sources (Telegram, Discord, Twitter)
GET /api/sources/:id
Get detailed information about a specific source
GET /api/sources/:id/sentiment
Get sentiment aggregates for a specific source
Base URL: http://localhost:3000
Response Format: JSON
Example: curl http://localhost:3000/api/health
